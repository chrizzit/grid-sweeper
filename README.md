# Grid Sweeper 🚩

A modern, premium grid-clearing game built with TypeScript, Vite, and PWA technology. This game is designed to be sleek, responsive, and installable on both desktop and mobile devices.

## Features

- **Modern UI**: Dark mode support with a premium, glassmorphism-inspired design.
- **PWA Support**: Install the game as a standalone app on your computer or phone.
- **Offline Play**: Play anywhere, even without an internet connection.
- **Personal Best Times**: Tracks your fastest wins locally in your browser.
- **Multiple Difficulties**: Beginner, Intermediate, and Expert levels.

## Installation (PWA)

To install Grid Sweeper:

1. Visit the live site in your browser.
2. Click the **"Install App"** icon in the address bar (Chrome/Edge desktop) or select **"Add to Home Screen"** from the browser menu (iOS/Android).

## Local Development

If you want to run this project locally:

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Run automated tests
npm test

# Build for production
npm run build
```
